# Dungeon Explorer

A text-based dungeon adventure game built in Python.

## Features
- Explore connected rooms
- Fight enemies
- Pick up and use items
- Manage health and inventory
- Win by defeating the final boss

## How to Run
1. Install Python 3
2. Download or clone this repository
3. Run the game:

   python game.py
